# 3D Gaussian Splatting for Real-Time Radiance Field Rendering

## 各向异性 3D 高斯作为场景表示

原论文不用体素网格或神经网络存整场，而用 \(N\) 个三维高斯原语 \(G_n\) 近似辐射场。每个高斯带有：

- 中心位置 \(\mathbf{p}_n\)；  
- 不透明度 \(\alpha_n\)（优化时用 logit，渲染前 sigmoid）；  
- 形状：用三维缩放 \(\mathbf{s}_n\)（对数空间优化）和单位四元数 \(\mathbf{r}_n\) 构造协方差  
  \(\Sigma_n = R(\mathbf{r}_n)\, S(\mathbf{s}_n)\, S(\mathbf{s}_n)^\top R(\mathbf{r}_n)^\top\)，  
  允许沿表面拉长、沿法向压扁，即各向异性，比各向同性球点更贴几何；  
- 外观：球谐系数（SH）表示视角相关颜色——先用 0 阶（漫反射色），训练中逐步升到 3 阶，共 48 维，以较低成本表达高光等视角效应。

渲染时沿用 EWA Splatting 思路：把 3D 高斯投影到像平面得到 2D 高斯 footprint，在像素 \(\mathbf{x}\) 上的贡献为

\[
\alpha_n(\mathbf{x}) = \alpha_n \exp\!\left(-\tfrac{1}{2}(\mathbf{x}-\mathbf{p}'_n)^\top \Sigma'_n{}^{-1}(\mathbf{x}-\mathbf{p}'_n)\right),
\]

再按深度排序做 α 混合得到颜色（与 NeRF 体渲染同族，但只在有原语覆盖的像素上计算）。空区域没有高斯，因此避免 NeRF 式全域密集采样。

## 交错优化与自适应密度控制

训练约 30k 迭代，损失为 L1 与 D-SSIM 的加权和。初始化：COLMAP 稀疏点云中每个 SfM 点对应一个高斯，颜色取该点邻域图像的 SH 初值，尺度由最近邻距离启发式设定。

原论文在「优化高斯参数」与「改变高斯个数」之间交替进行（interleaved）：

| 操作 | 触发条件（原论文） | 作用 |
|------|-------------------|------|
| Clone（克隆） | 视空间位置梯度大，且该高斯尺度较小 | 复制一个高斯，细化欠采样区域 |
| Split（分裂） | 视空间位置梯度大，且该高斯尺度较大 | 拆成两个更小高斯，替换原高斯 |
| Prune（剪枝） | 不透明度低于阈值，或屏幕 footprint 过大 | 删除几乎透明或异常大的点 |

梯度在每隔若干步统计：对可见高斯累积 \(\partial \mathcal{L}/\partial \mathbf{p}\) 在图像平面上的范数，梯度大的地方说明重建误差仍大，需要更多原语。致密化阶段结束后还会周期性重置不透明度，避免大量高斯卡在「半透明」状态占着不贡献。

此外，SH 阶数每 1000 步提高一档，先学粗颜色再学视角相关细节。最终常得到 \(10^5\)～\(10^7\) 量级高斯（场景越复杂越多）。

## 可见性感知的 tile-based 可微光栅化

若对每个像素遍历全部百万高斯，实时渲染不可行。原论文实现 CUDA 光栅化器，流程大致为：

1. 视锥剔除：丢弃相机看不到的高斯；  
2. 投影：用视图变换与 Jacobian 得到 2D 均值 \(\mathbf{p}'_n\) 与 2D 协方差 \(\Sigma'_n\)；  
3. 按屏幕 tile（小块）划分，只把与 tile 相交的高斯列入该 tile 的列表；  
4. 每个 tile 内按深度排序，对覆盖像素做 α 混合；  
5. 整条管线可微，渲染 loss 的梯度可回传到 \(\mathbf{p}, \mathbf{s}, \mathbf{r}, \alpha\)、SH 等参数。

「可见性感知」指：高斯只在与自身 footprint 相交的 tile / 像素上参与计算，复杂度近似与可见原语数相关，而非场景总点数。论文在 1080p 上可达约 100+ FPS 量级推理，训练也因快速反传而可在数十分钟内收敛（视数据集与 GPU 而定）。
