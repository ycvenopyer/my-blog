## System Prompt vs User Prompt

| 类型 | 作用 | 范围 | 遵循优先级 | 类比 | 示例 |
|------|------|------|------|------|------|
| System prompt | 定义AI的人设、行为准则和长期记忆 | 全局、持久（整个会话或应用） | P0（最高） | 公司章程、角色剧本、操作系统 | “你是一位严谨的法律顾问。所有回答必须引用具体发条，并以‘免责声明’结尾” |
| User Prompt | 下达具体、临时的任务指令 | 局部、即时（单词问答） | P1（次高） | 领导分配的当日任务 | “帮我分析一下这份租赁合同中潜在风险” |

System Prompt就是演员的角色剧本和导演的总纲。它规定了演员的性格（是霸道总裁还是邻家小妹儿）、背景（是来自火星还是来自中世纪）、知识领域（是懂代码还是懂历史）和表演风格（是严肃还是搞笑）。

User Prompt 则是导演在每一幕给出的具体指令，比如“现在，给我一个愤怒的表情”、“跟观众解释一下你为什么这么做”。

System Prompt给AI搭了一个框架，让它有了一个稳定的“人格”。而User Prompt则是在这个框架里，驱动AI去完成一个个具体的任务。
