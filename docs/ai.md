# AI Time

如果说从2000年开始互联网领域开始爆发，那么从2022年开始就是AI大模型领域的爆发，约莫20年的变革足以改变行业内的版图。计算机科学的历史其实并不长，但是发展的速度越来越快。早期从机器学习到深度学习，训练方式还处于数据集的特征处理和信息抽取，未能发现隐藏机制和深层信息，自然语言处理还在传统的word2vec，计算机视觉还是ResNet的天下，2017年的一篇《Attention is All You Need》轰动全球，OpenAI推出的GPT系列更是改变了传统计算机与软件工程的发展方向。如今从以年为单位工程迭代（Prompt Engineering -> Context Engineering -> Harness Engineering），到以月为单位的模型迭代（26年一季度各家公司已经推出若干旗舰模型），到以周为单位的产品迭代（产品设计方案和产品原型层出不穷，个人智能体兴起，AGI加速演进），不少人直言“只要学的慢什么都不用学”。

AI时代到底学什么？怎么学？我正在逐步探索这个问题。这个板块包含AI模型基座到AI应用，是我的AI知识库。所有文章内容除个人思考总结、各种渠道摘录外，还包括AI工具生成内容，请仔细甄别。
