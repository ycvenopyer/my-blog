## 🚀 个人项目

### 1. 个人博客系统

**技术栈**: Python + Flask + Markdown

**简介**:
- 基于 Flask 开发的轻量级博客系统
- 支持 Markdown 文章编辑
- 响应式设计，移动端友好

**主要功能**:
- 文章发布与分类
- 评论系统
- 标签管理
- 全文搜索

**项目链接**: [GitHub](https://github.com/ycvenopyer/my-blog)

### 3. 命令行工具集

**技术栈**: Python + Click

**包含工具**:
- 文件批量重命名
- 目录结构生成器
- 图片压缩工具
- 代码格式化工具

**安装使用**:

```bash
pip install my-tools
my-tools rename --pattern "IMG_{n}"
my-files tree --output structure.txt
```
